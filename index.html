HTML Lists

Structuring content effectively with ordered, unordered, and definition lists.

Ordered Lists <ol>

Use ordered lists for items where sequence is important, like steps in a recipe or a ranked list.

Example Output:

  1. First, preheat the oven.
  2. Second, mix the ingredients.
  3. Third, bake for 20 minutes.

HTML Code:

<ol>
  <li>First, preheat the oven.</li>
  <li>Second, mix the ingredients.</li>
  <li>Third, bake for 20 minutes.</li>
</ol>

Unordered Lists <ul>

Unordered lists are perfect for items without a specific sequence, like a list of features or ingredients.

Example Output:

  • Apples
  • Bananas
  • Oranges

HTML Code:

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>

Definition Lists <dl>

Definition lists are ideal for glossaries or displaying key-value pairs, clearly associating terms with their descriptions.

Example Output:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets

HTML Code:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
