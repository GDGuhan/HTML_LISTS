Embedding Images & Media in HTML

Learn how to bring your web pages to life with images and interactive maps.

The `<img>` Tag

The `<img>` tag is used to embed an image. It requires the `src` attribute for the image path and the `alt` attribute for accessibility. 

<img src="path/to/your/image.jpg" 
     alt="A descriptive text for the image" 
     width="500" 
     height="300">

Image Maps

Image maps create clickable areas on an image using `<map>` and `<area>` tags, linking different parts of the image to various URLs. 

<img src="workplace.jpg" alt="Workplace" usemap="#workmap">
<map name="workmap">
  <area shape="rect" coords="34,44,270,350" href="computer.htm">
  <area shape="circle" coords="337,300,44" href="coffee.htm">
</map>
